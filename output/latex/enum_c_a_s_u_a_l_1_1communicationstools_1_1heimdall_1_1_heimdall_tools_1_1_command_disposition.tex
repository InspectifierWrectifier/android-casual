\subsection{Detailed Description}
Status for decision making based on return value from heimdall.

S\-U\-C\-C\-E\-S\-S\-: Heimdall executed sucessfully

H\-A\-L\-T\-E\-D\-: Heimdall encountered a failure which cannot be recovered.

C\-O\-N\-T\-I\-N\-U\-E\-: Heimdall did not execute sucessfully, but we can try again.

M\-A\-X\-I\-M\-U\-M\-R\-E\-T\-R\-I\-E\-S\-E\-X\-C\-E\-E\-D\-E\-D\-: We've tried to continue four times now.
